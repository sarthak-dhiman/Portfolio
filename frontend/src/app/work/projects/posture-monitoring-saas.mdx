## Overview

An end-to-end SaaS platform designed for ergonomic monitoring. It features a robust Django backend with WebSocket support for real-time communication and a PyQt6-based Windows client for on-device YOLOv8-Pose inference.

## Key Features

- **Real-time Inference**: YOLOv8-Pose model integrated into a desktop client for zero-latency monitoring.
- **Scalable Backend**: Django-based management system with PostgreSQL and WebSockets.
- **Monetization**: Integrated Razorpay for seamless subscription and billing management.
- **Native Experience**: Custom-built PyQt6 Windows application for background operation.

## Technologies Used

- **Python**
- **Django**
- **WebSockets**
- **YOLOv8**
- **PostgreSQL**
- **Razorpay**
- **PyQt6**

## Impact

Successfully bridged the gap between complex AI models and a commercial product, delivering a native desktop experience with automated billing and real-time monitoring capabilities.
