## Overview

HeliosPredict is a forecasting engine for solar energy output based on historical irradiance and plant telemetry, hosted on AWS EC2.

## Key Features

- **Deep Learning**: Full TensorFlow pipeline for high-precision energy forecasting.
- **Real-time Data**: Integrated live meteorological data via WeatherAPI.
- **Cloud Hosted**: Deployed on AWS EC2 for scalable and reliable performance.

## Technologies Used

- **Python**
- **TensorFlow**
- **Pandas**
- **AWS EC2**
- **WeatherAPI**

## Impact

Authored a full TensorFlow pipeline and integrated live meteorological data for real-time prediction, enabling more efficient solar farm management.
