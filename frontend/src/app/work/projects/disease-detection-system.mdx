## Overview

A multi-model medical diagnostic API designed to identify jaundice, skin conditions across 38 classes, and nail pathology. The system leverages asynchronous processing to ensure high performance even under heavy GPU-intensive tasks.

## Key Features

- **Multi-Model Support**: Integrated models for diverse medical diagnostic tasks.
- **Asynchronous Architecture**: Built with FastAPI, Celery, and Redis to handle concurrent inference requests.
- **High Accuracy**: Optimized models achieving 94% accuracy in jaundice and 96% in nail pathology detection.
- **Production-Ready**: Fully dockerized and ready for cloud deployment.

## Technologies Used

- **Python**
- **FastAPI**
- **PyTorch**
- **HuggingFace**
- **Celery & Redis**
- **Docker**

Developed a robust diagnostic tool that provides rapid, accurate medical insights, demonstrating the potential of AI in augmenting healthcare workflows.
